# homebrew-macrift

Homebrew tap for [macrift](https://github.com/emylfy/macrift) — an interactive macOS setup TUI.

```sh
brew install emylfy/macrift/macrift
# or
brew tap emylfy/macrift
brew install macrift
```

`Formula/macrift.rb` is generated on each release by `scripts/release-assets.sh` in the
main repo — do not hand-edit it.
